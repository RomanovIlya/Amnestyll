# Amnestyll
>  Amnestyll — психологический триллер в мрачном сеттинге, где игрок исследует фрагменты памяти героя и ищет пропавшую жену. Предмет симуляции — состояние сознания, сомнение и расследование личной тайны. Игра ориентирована на аудиторию 12–35 лет, любящую атмосферные сюжетные проекты с mystery-элементами. Платформа — PC (Steam, VK play).
